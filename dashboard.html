Migration des Secrets vers Vault

L'injecteur Vault est activé. Pour migrer un secret :

  1. Créez le secret dans Vault : vault kv put secret/minio rootUser="admin" ...
  2. Ajoutez les annotations au déploiement : 
    vault.hashicorp.com/agent-inject: "true"
vault.hashicorp.com/role: "app-role"
vault.hashicorp.com/agent-inject-secret-config: "secret/data/app"
